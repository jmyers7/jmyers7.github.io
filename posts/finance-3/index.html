Equivalent martingale measures; the Fundamental Theorem of Asset Pricing

Theorem 1 (Fundamental Theorem of Asset Pricing) A model is arbitrage-free if and only if there exists an equivalent martingale measure.

NoteProof.

The proof for the binomial model was presented above, so we give a proof for the trinomial model.

First, suppose that \(Q\) is an EMM for the trinomial model with risk-free gross return \(R\), up-factor \(u\), middle-factor \(m\), and down-factor \(d\). Then \(Q\) is determined by three probabilities \(q_u\), \(q_m\), and \(q_d\) which are solutions to the system of linear equations

\[ \begin{align*} q_u + q_m + q_d &= 1 \\ uq_u + mq_m + dq_d &= R. \end{align*} \tag{1}\]

Indeed, the necessity of the first equation is obvious, while the second equation guarantees the martingale property for the discounted prices. In addition to the linear system, we also have the constraints

\[ 0 \leq q_u, q_m, q_d \leq 1. \tag{2}\]

We solve the linear system to obtain

\[ \begin{align*} q_u &= \frac{m-d}{u-m} q_d + \frac{R-m}{u-m}, \\ q_m &= \frac{d-u}{u-m} q_d + \frac{u-R}{u-m}, \end{align*} \tag{3}\]

expressing \(q_u\) and \(q_m\) in terms of the free parameter \(q_d\). Using these formulas, the reader may show that, provided

\[ \frac{m-R}{m-d} \leq q_d \leq \frac{u-R}{m-d}, \tag{4}\]

we have \(0 \leq q_u \leq 1\). Likewise, provided that

\[ \frac{m-R}{u-d} \leq q_d \leq \frac{u-R}{u-d}, \tag{5}\]

we have \(0 \leq q_m \leq 1\). Since \(m < u\), the single set of inequalities

\[ \frac{m-R}{m-d} \leq q_d \leq \frac{u-R}{u-d} \tag{6}\]

implies both (4) and (5). But we must have \(0 \leq q_d \leq 1\) as well, and so we must have both

\[ \frac{m-R}{m-d} \leq 1 \quad \text{and} \quad 0 \leq \frac{u-R}{u-d}. \]

The first inequality implies \(R\geq d\), while the second implies \(R \leq u\). Thus, the model is arbitrage-free.

Conversely, suppose that the model is arbitrage-free, so that \(d\leq R \leq u\). Then we may select a parameter \(q_d\) so that

\[ \max \left( \frac{m-R}{m-d}, 0 \right) \leq q_d \leq \frac{u-R}{u-d}. \]

The no-arbitrage condition guarantees that \(0 \leq q_d \leq 1\). Then the inequalties (6) are satisfied, and hence so too are the inequalities (4) and (5). If we then define \(q_u\) and \(q_m\) by (3), we obtain solutions to the system (1) satisfying the constraints (2). Thus, we have shown that an EMM exists.

For the reader who did not read the proof, the gist is this. Let \(R\) be the risk-free gross return of an arbitrage-free trinomial model, and let \(u\), \(m\), and \(d\) be the up-, middle-, an down-factors, respectively. Then the no-arbitrage condition \(d\leq R \leq u\) implies the second two inequalities in

\[ 0 \leq \max \left( \frac{m-R}{m-d}, 0 \right) \leq \frac{u-R}{u-d} \leq 1. \tag{7}\]

Then, the proof shows that there is a one-parameter family of EMMs for the trinomial model, parametrized by a number \(q\) between the middle two numbers in (7), namely

\[ \max \left( \frac{m-R}{m-d}, 0 \right) \leq q \leq \frac{u-R}{u-d}. \]

If we write \(a\) for the left-hand side of this interval, and \(b\) for the right-hand side, then we may reparametrize using the map

\[ \theta \mapsto q= (b-a)\theta + a, \]

and so our family of EMMs is parametrized by \(\theta\in [0,1]\).
