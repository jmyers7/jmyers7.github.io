Definition 1 Let \(S\) be a set and let \(\mathcal{R}\) be a nonempty set of subsets of \(S\). Then \(\mathcal{R}\) is called a ring if it is closed under binary unions and relative complements.

Definition 2 Let \(S\) be a set and let \(\mathcal{S}\) be a set of subsets of \(S\). Then \(\mathcal{S}\) is called a semi-ring if it contains the empty set, it is closed under binary intersections, and if every relative complement of sets in \(\mathcal{S}\) is a finite union of disjoint sets in \(\mathcal{S}\).

Theorem 1 Let \(\mathcal{R}\) be a ring of sets in a set \(S\). Then \(\mathcal{R}\) is a semi-ring.

Proof. Choosing an \(A\in \mathcal{R}\), note that \(\emptyset = A\smallsetminus A\), so that \(\mathcal{R}\) contains the empty set. Since \[ A\cap B = A\smallsetminus (A\smallsetminus B) \] for \(A,B\in \mathcal{R}\), we see that \(\mathcal{R}\) is closed under binary intersections. Finally, since \(\mathcal{R}\) is itself closed under relative complements, it satisfies the last property for a semi-ring. Q.E.D.

For the next definition, we suppose that we have \((S,\mathcal{B}_S)\), where \(S\) is a topological space and \(\mathcal{B}_S\) is its Borel \(\sigma\)-field. If \(A\subset S\) is a topological space subspace, then it has its own Borel \(\sigma\)-field \(\mathcal{B}_A\) generated by the subspace topology inherited from \(S\). Since this topology guarantees that the inclusion map \(iota:A\to S\) is continuous, the inclusion map is also measurable. Thus, we have that \(\iota^{-1}\mathcal{B}_S \subset \mathcal{B}_A\), since the first \(\sigma\)-field is the coarest onen for which \(\iota\) is measurable. However, every relative open set \(U\cap A\), for \(U\subset S\) open, is contained in \(\iota^{-1}\mathcal{B}_S\) because \(U\in \mathcal{B}_S\) and \(U\cap A = \iota^{-1}U\). But \(\mathcal{B}_A\) is generated by all sets of the form \(U\cap A\), and hence we have \(\mathcal{B}_A \subset \iota^{-1}\mathcal{B}_S\). Thus, these two \(\sigma\)-fields are equal.

Definition 3 Let \((S,\mathcal{S})\) and \((T,\mathcal{T})\) be two measurable spaces.

  1. An isomorphism from \((S,\mathcal{S})\) to \((T,\mathcal{T})\) is a bijective function \(f:S\to T\) for which both \(f\) and \(f^{-1}\) are measurable.

  2. The measurable space \((S,\mathcal{S})\) is called a Borel space if it is isomorphic to a Borel set in \([0,1]\), where the latter is equipped with its Borel \(\sigma\)-field generated its subspace topology.

For example, the real line \(\mathbb{R}\) equipped with its Borel \(\sigma\)-field is a Borel space, since it is homeomorphic to the open interval \((0,1)\). In fact, every euclidean space \(\mathbb{R}^k\) is a Borel space, which follows from the next important result. To state it, we first recall that a topological space is called a Polish space if it is separable and its topology can be generated by a complete metric. Thus, each \(\mathbb{R}^k\) is a Polish space.

Theorem 2 Every Polish space is a Borel space.

Definition 4 Let \((S,\mathcal{S})\) be a Borel space. A localizing sequence is a sequence of sets \((S_n)_{n=1}^\infty\) in \(\mathcal{S}\) for which \(S_n \uparrow S\). Given such a localizing sequence, a subset \(B\subset S\) is called bounded (relative to the given localizing sequence) if \(B\subset S_n\) for some \(n\). We write \(\hat{\mathcal{S}}\) for the set of bounded sets in \(\mathcal{S}\). A localized Borel space is a Borel space equipped with a choice of localizing sequence.

The Borel space \((\mathbb{R},\mathcal{B}_\mathbb{R})\) can be equipped with a localizing sequence given by the open balls centered at \(0\): \[ B(0;1) \subset B(0;2) \subset \cdots \subset B(0;n) \subset \cdots \subset \mathbb{R}. \]

Definition 5 Let \(S\) be a topological space and suppose that the pair \((S,\mathcal{B}_S)\) is a localized Borel space. A class \(\mathcal{C}\subset \hat{\mathcal{B}}_S\) is dissecting if every open set \(U\) is a countable union of sets in \(\mathcal{C}\), and every set \(B\in \hat{\mathcal{B}}_S\) is covered by finitely many sets in \(\mathcal{C}\).

Theorem 3 Let \((S_n)_{n=1}^\infty\) be a localizing sequence for a Borel space \((S,\mathcal{S})\). Then the set \(\hat{\mathcal{S}}\) is a ring.

Definition 6 A measure \(\mu\) on a localized Borel space \((S,\mathcal{S})\) is called locally finite if \(\mu B<\infty\) for all \(B\in \hat{\mathcal{S}}\). We write \(\mathcal{M}_S\) for the class of all such measures.

Let \(\mathcal{M}_S\) be the set of locally finite measures on a localized Borel space \((S,\mathcal{S})\). For each \(B\in \mathcal{S}\), we have the evaluation map \[ \pi_B: \mathcal{M}_S \to \overline{\mathbb{R}}, \quad \mu \mapsto \mu B. \] We equip \(\mathcal{M}_S\) with the \(\sigma\)-field generated by these maps, as \(B\) varies over all measureable sets \(B\in \mathcal{S}\). This \(\sigma\)-field is the smallest one for which each \(\pi_B\) is measurable.
