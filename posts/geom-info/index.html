Introduction

Algebras as information carriers

An observer monitors the result of some random experiment. The set of all possible results is collected into a sample space \(\Omega\), whose elements \(\omega\) are called outcomes or sample points. Suppose that the experiment is finished, so that a definite \(\omega_0\) in \(\Omega\) is produced, but that the exact identity of \(\omega_0\) remains unknown to the observer. However, suppose that the observer has access to a set \(\mathcal{F}\) of subsets of \(\Omega\) with the following property:

For each \(A\in \mathcal{F}\), the observer can decide whether \(\omega_0\in A\) or not.

The observer’s decisions reflect the information they possess, so we can treat the class \(\mathcal{F}\) as a proxy for that information. Sets in \(\mathcal{F}\) are called (decidable) events. If the observer confirms that \(\omega_0\in A\) for some \(A\in \mathcal{F}\), then we say that “the event \(A\) has occurred”.

It is never in question that \(\omega_0\in \Omega\), so it is natural to include \(\Omega\) in the class \(\mathcal{F}\) of decidable events. But what other properties must \(\mathcal{F}\) have?

  1. If the observer can determine whether an event \(A\) has occurred, then they obviously can determine whether \(A\) did not occur. This means that if \(A\in \mathcal{F}\), then \(A^c \in \mathcal{F}\) too. (Here, \(A^c\) is the absolute complement of \(A\), consisting of those \(\omega\in \Omega\) that are not in \(A\).)

  2. If the observer can determine separately whether events \(A\) and \(B\) have occurred, then they can determine whether \(A\) or \(B\) has occurred. Since \(A\cup B\) consists of those outcomes that are either in \(A\) or \(B\), this means that \(A\cup B\) must be in \(\mathcal{F}\) when both \(A\) and \(B\) are.

  3. Similarly, if the observer can determine separately whether events \(A\) and \(B\) have occurred, then they can determine whether \(A\) and \(B\) has occurred. This implies that \(\mathcal{F}\) must contain the intersection \(A\cap B\) whenever it contains both \(A\) and \(B\).

These three properties capture what it means for our collection \(\mathcal{F}\) of events to represent a logically coherent state of information. They lead us naturally to the formal definition of an algebra of sets.

Actually, we need not explicitly require that \(\mathcal{F}\) contains all (binary) intersections of events, since this follows from properties (1) and (2).

Definition 1 Let \(\mathcal{F}\) be a set of subsets of a set \(\Omega\). Then \(\mathcal{F}\) is called an algebra of sets (in \(\Omega\)) if it has the following properties:

  1. It contains \(\Omega\).

  2. It is closed under (absolute) complementation:

\[ A\in \mathcal{F}\quad \Rightarrow \quad A^c\in \mathcal{F}. \]

  1. It is closed under binary unions:

\[ A_1, A_2\in \mathcal{F}\quad\Rightarrow \quad A_1 \cup A_2 \in \mathcal{F}. \]

We have thus argued that our class \(\mathcal{F}\) of decidable events—encoding our observer’s information—must be an algebra of sets in the sample space \(\Omega\).

As we indicated in the margin note above, algebras of sets are also closed under binary intersections. Moreover, an easy inductive proof shows that algebras are also closed under all finite unions and intersections, not just binary ones. In this post we’ll stick to finite sample spaces, but in the infinite case one usually strengthens these closure properties to include countable unions, yielding \(\sigma\)-algebras of sets.

Example: coin tosses

We consider the sample space \(\Omega\) consisting of all binary sequences of length 3:

\[ \Omega = \{000, 001, 010, 011, 100, 101, 110, 111\}. \]

We imagine that a binary sequence \(\omega\) in \(\Omega\) represents the result of flipping a coin three times in succession, with \(0\) when a tail occurs, and \(1\) when a head is shown. We let \(X_i(\omega)\) denote the result of the \(i\)-th flip in \(\omega\) (so \(X_i(\omega)=0\) or \(1\)), and we write

\[ Y = X_1 + X_2 + X_3. \]

The four functions \(X_1\), \(X_2\), \(X_3\), and \(Y\) are, of course, random variables on \(\Omega\). If the coin has probability \(\theta\) of showing heads, then

\[ X_i \sim \mathcal{B}er(\theta) \quad \text{for each $i$, and} \quad Y\sim \mathcal{B}in(3,\theta). \]

We will be carrying out some basic computations later, so let’s implement our scenario in the machine, by coding a data frame in which each row represents an outcome \(\omega\):

Code 
Omega <- data.frame(
  X1 = c(0, 0, 0, 0, 1, 1, 1, 1),
  X2 = c(0, 0, 1, 1, 0, 0, 1, 1),
  X3 = c(0, 1, 0, 1, 0, 1, 0, 1)
)

Omega <- Omega %>% 
  mutate(Y = X1 + X2 + X3)

Now, suppose that the coin was flipped three times, producing a definite outcome \(\omega_0\) in \(\Omega\), but that its exact identity is unknown to our observer. Suppose that the only piece of information that the observer has is whether the first flip produced a tail or head, i.e., the observer can decide whether \(X_1(\omega_0) = 0\) or \(1\). This information may be encoded in an algebra of sets \(\mathcal{F}_1\).

Indeed, the observer can decide which of the two sets

\[ A_1 = \{000,001,010,011\} \quad \text{or} \quad A_2 = \{100,101,110,111\} \]

contains \(\omega_0\), since the first set \(A_1\) consists of those \(\omega\) with \(X_1(\omega)=0\), while the second contains the outcomes with \(X_1(\omega)=1\). As you can easily check, the set

\[ \mathcal{F}_1 = \{\emptyset, A_1, A_2, \Omega\} \]

is an algebra of sets in \(\Omega\). Let’s insert a column in our data frame tracking the membership of each outcome \(\omega\) (i.e., row) in \(A_1\) or \(A_2\):

Code 
Omega <- Omega %>% 
  group_by(X1) %>% 
  mutate(Ai = cur_group_id()) %>% 
  ungroup() %>% 
  as.data.frame()

Suppose now, after having learned the outcome of the first flip in \(\omega_0\), that the observer is given the identity of the second flip as well. Then, the total information held by our observer (i.e., the outcomes of the first two flips) is encoded in a second algebra \(\mathcal{F}_2\) that contains the first:

\[ \mathcal{F}_1 \subset \mathcal{F}_2. \]

Inclusions of algebras represent refinements of information.

The new algebra \(\mathcal{F}_2\) must contain the sets

\[ B_1 = \{000,001\}, \quad B_2 = \{010,011\}, \quad B_3 = \{100,101\}, \quad B_4 = \{110,111\}, \]

since deciding whether \(\omega_0\in B_i\) for each \(i\) will identify the first two flips in \(\omega_0\). However, in order to be an algebra, \(\mathcal{F}_2\) must contain more than just these sets since it must be closed under complements and unions. We compute:

\[ \mathcal{F}_2 = \{\emptyset, B_1, B_2, B_3, B_4, A_1, A_2, B_1\cup B_3, B_1\cup B_4, B_2\cup B_3, B_2\cup B_4, \Omega \}. \]

Note that \(\mathcal{F}_2\) does, in fact, contain \(\mathcal{F}_1\). Let’s insert our new algebra into the machine:

Omega <- Omega %>%

Finally, after having learned the outcomes of the first two flips in \(\omega_0\), our observer is given the outcome of the third and last flip, completely specifying \(\omega_0\) and representing complete information. The sum total of our observer’s information is then encoded in a third algebra \(\mathcal{F}_3\) that refines the information in \(\mathcal{F}_2\) (and thus also \(\mathcal{F}_1\)):

\[ \mathcal{F}_1 \subset \mathcal{F}_2 \subset \mathcal{F}_3. \]

The algebra \(\mathcal{F}_3\) must contain the eight singletons

\[ \begin{gather*} C_1 = \{000\}, \ C_2 = \{001\}, \ C_3 = \{010\}, \ C_4 = \{011\}, \\ C_5 = \{100\}, \ C_6 = \{101\}, \ C_7 = \{110\}, \ C_8 = \{111\}, \end{gather*} \]

since deciding whether \(\omega_0\in C_i\) for each \(i\) means identifying \(\omega_0\) exactly. But, as with \(\mathcal{F}_2\), the class \(\mathcal{F}_3\) must contain more than just these eight sets if it is to be an algebra. In fact, \(\mathcal{F}_3\) must be the power set on \(\Omega\), i.e., the class of all subsets of \(\Omega\). We insert one last column in our data frame for the \(C_i\)’s:

Omega <- Omega %>% 
  group_by(X1, X2, X3) %>% 
  mutate(Ci = cur_group_id()) %>% 
  ungroup() %>%

Note that the \(A_i\) partition the sample space \(\Omega\), in the sense that \(\Omega = A_1 \cup A_2\) and \(A_1\) and \(A_2\) are disjoint. Similarly, the \(B_i\) and \(C_i\) also partition \(\Omega\), and they represent progressively finer partitions, in the sense that each \(A_i\) is a union of \(B_i\)’s, and each \(B_i\) is a union of \(C_i\)’s. The \(A_i\), \(B_i\), and \(C_i\) are also the atoms in their respective algebras, in the sense that no \(A_i\) (respectively, \(B_i\) or \(C_i\)) properly contains a nonempty subset of \(\mathcal{F}_1\) (respectively, \(\mathcal{F}_2\) or \(\mathcal{F}_3\)).

Note also that two outcomes \(\omega_1\) and \(\omega_2\) in the same \(A_i\) are indistinguishable using only the information in \(\mathcal{F}_1\). Indeed, the only information our observer has is whether \(\omega \in A_i\) or not, and hence if \(\omega_1\) and \(\omega_2\) are in the same \(A_i\), then the observer cannot determine whether \(\omega_1\) and \(\omega_2\) are distinct outcomes. The same is true about two outcomes in the same \(B_i\), though the \(C_i\) do not exhibit this ambiguity since \(\mathcal{F}_3\) represents complete information.

Each algebra in the chain of inclusions

\[ \mathcal{F}_1 \subset \mathcal{F}_2 \subset \mathcal{F}_3 \]

captures a distinct level of information about the outcome \(\omega_0\). This nested structure can be viewed as a simple model of learning over time: as the experiment unfolds and the observer acquires new data, their algebra of decidable events expands. Sets of indistinguishable outcomes split apart into finer partitions—until, at last, the algebra \(\mathcal{F}_3\) corresponds to complete knowledge of \(\omega_0\).

Indistinguishability and partitions

In this section, we formalize these observations regarding indistinguishability and partitions, and we state and prove a pair of useful theorems. Let \(\Omega\) be any set (finite or not) and \(\mathcal{F}\) an algebra of sets in \(\Omega\). Indistinguishability of two elements \(\omega\) and \(\omega_0\) of \(\Omega\) (relative to \(\mathcal{F}\)) occurs exactly when we have

\[ \omega \in A \quad \Leftrightarrow \quad \omega' \in A, \quad \forall A\in \mathcal{F}. \tag{1}\]

Elements \(\omega\) and \(\omega'\) related in this way will be called \(\mathcal{F}\)-equivalent.

This notion of equivalence is evidently an equivalence relation in the technical sense, and thus we obtain a partition of \(\Omega\) into equivalence classes. The next theorem describes some properties of these equivalence classes.

Theorem 1 (Properties of indistinguishability equivalence classes) Let the notation be as above, and write \([\omega]\) for the equivalence class of \(\omega\) under the equivalence relation (1).

  1. The equivalence class \([\omega]\) is the intersection of all \(A\in \mathcal{F}\) that contain \(\omega\): \[ [\omega] = \bigcap_{\omega \in A \in \mathcal{F}} A. \tag{2}\]

  2. If \([\omega]\) contains a nonempty subset \(A\) which is also in \(\mathcal{F}\), then \(A = [\omega]\).

Proof. (1): The goal is to establish the equality (2) of sets. The subset containment “\(\subset\)” is more or less trivial, so we focus on the opposite containment “\(\supset\)”. For this, suppose that \(\omega'\) is an element in the intersection in (2); we need to prove that (1) holds for all \(A\in \mathcal{F}\). But if \(\omega \in A\), then \(\omega'\in A\) as well since \(A\) appears in the intersection in (2). Conversely, suppose that \(\omega'\in A\), but that \(\omega \in A^c\). Then, since \(A^c\in \mathcal{F}\), the complement \(A^c\) appears in the intersection in (2), and hence \(\omega' \in A^c\). However, this contradicts the assumption that \(\omega'\in A\).

(2): If \(A\) contained \(\omega'\), then \(\omega' \in A \subset [\omega]\), and hence \(\omega\) and \(\omega'\) are \(\mathcal{F}\)-equivalent. But then \(\omega\in A\) as well, which—in view of (1)—implies that \([\omega] \subset A\). Thus \(A =[\omega]\). Q.E.D.

In particular, part (2) of the theorem shows that if an equivalence class \([\omega]\) is in \(\mathcal{F}\) (which is the case if \(\Omega\) is finite), then the only sets in \(\mathcal{F}\) that are contained in \([\omega]\) are itself and \(\emptyset\). In this sense, the equivalence classes \([\omega]\) are the atoms of the algebra \(\mathcal{F}\), i.e., the “non-reducible” subsets of \(\mathcal{F}\). This notion is familiar to us from the \(A_i\), \(B_i\) and \(C_i\) in the previous section.

So, every algebra \(\mathcal{F}\) of sets in \(\Omega\) determines a partition of \(\Omega\). Conversely, every partition \(\mathfrak{P}\) of \(\Omega\) determines an algebra as the smallest algebra containing each set in \(\mathfrak{P}\); this algebra is denoted \(\sigma(\mathfrak{P})\).

Theorem 2 (Algebras \(\Leftrightarrow\) Partitions) Let \(\mathcal{F}\) be an algebra of sets in \(\Omega\), and assume that each \(\mathcal{F}\)-equivalence class \([\omega]\) is in \(\mathcal{F}\). If \(\mathfrak{P}\) is the partition induced by these equivalence classes, then \(\sigma(\mathfrak{P}) = \mathcal{F}\).

Proof. Since \(\mathcal{F}\) is an algebra containing each set in \(\mathfrak{P}\), we must have \(\sigma(\mathfrak{P}) \subset \mathcal{F}\). On the other hand, if \(A\in \mathcal{F}\) and \(P\in \mathfrak{P}\), then the intersection \(A\cap P\) is both a subset of \(P\) and a set in \(\mathcal{F}\). But since \(P\) is atomic, we must have either \(A\cap P = \emptyset\) or \(A\cap P = P\). The latter implies that \(P\subset A\). This should imply that \(A\) is a union of \(P\)’s and thus is in \(\sigma(\mathfrak{P})\). Q.E.D.

Finally, if \(\mathcal{F}\) and \(\mathcal{G}\) are two algebras of sets in \(\Omega\), then we say \(\mathcal{F}\) refines \(\mathcal{G}\) if \(\mathcal{G}\subset \mathcal{F}\). With two algebras, there are two partitions of \(\Omega\) and two types of equivalence classes: we write \([\omega]_\mathcal{F}\) and \([\omega]_\mathcal{G}\) for the equivalence class of \(\omega\in \Omega\) with respect to \(\mathcal{F}\)- and \(\mathcal{G}\)-equivalence, respectively.

Theorem 3 (Refinement Theorem) Let \(\mathcal{F}\) and \(\mathcal{G}\) be two algebras of sets in \(\Omega\). Then \(\mathcal{F}\) refines \(\mathcal{G}\) if and only if every equivalence class \([\omega]_\mathcal{G}\) is partitioned into a union of \(\mathcal{F}\)-equivalence classes.
